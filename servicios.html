Comandos GitHub

Primero se le indica a GIT que vamos a agregar un origen remoto de nuestros archivos:
git remote add origin https://github.com/[usuario_git]/[repositorio].git

git push origin main


Para envíar (PUSH:EMPUJAR) mis archivos locales, al remoto, se hace por medio de ese comando.

git pull origin main


Pero si eso no funciona, es porque primero se tiene que traer (PULL:JALAR) lo que está en el remoto al local.
