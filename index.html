Hello Git

Todo empieza con git init


Luego debes iniciar con git add «.» y luego deberás subir esa adición al repositorio local con un commit así: git commit -m "un mensaje"


git status

Revisa el estado y el lugar donde estás.

git commit -am "msj"

Sube al repositorio local sin necesidad de git add «.»

Esta parte es importante, por favor presta atención a los detalles.
Recuerda que para las nuevas ramas en GIT usas el comando git branch + "nombre de la rama".
Y luego pasas a la rama con git checkout + "nombre de la rama"


git log
